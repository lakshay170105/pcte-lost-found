# 🎓 PCTE Lost & Found Portal

A full-stack MERN application for managing lost and found items at PCTE Group of Institutes.

## 🚀 Quick Start

### Frontend
```bash
cd vite-project
npm install
npm run dev
# Opens at http://localhost:5173/
```

### Backend
```bash
cd backend
npm install
npm run dev
# Runs at http://localhost:5001/
```

## 🔧 Setup

1. **MongoDB:** Update password in `backend/.env`
2. **Email:** Already configured with Gmail
3. **Run:** Start both frontend and backend

## ✨ Features

- User authentication with JWT
- Report lost/found items
- Real-time updates (🔴 LIVE)
- Smart matching algorithm
- Email notifications
- Image upload support
- Admin dashboard
- Date validation (current year only)
